Consulta de medios o media query es una técnica CSS introducida en CSS3.

Utiliza la regla @media para incluir un bloque de propiedades de CSS sólo si una determinada condición es verdadera. @media captura la pantalla o el medio donde se está visualizando la página web. Dentro de esa captura puedo clasificar el ancho de la pantalla. Boostrap clasifica 4 puntos de quiebre para saber en qué dispositivo se está visualizando.

  • Extra small devices: Phones (<768px) Ancho de pantalla menor hasta 767px
  • Small devices: Tablets (>=768px) Ancho de pantalla desde 768px hasta 991px
  • Medium devices: Desktops (>=992px) Ancho de pantalla desde 992px hasta 1200px
  • Large devices: Desktops (>=1200px) Ancho de pantalla desde 1200px

En desktop un standar es: 1368px En una tablet posición horizontal es: 1024px En una tablet posición vertical es: 768px En un movil en posición horizontal: desde 480px hasta más En un movil en posición vertical: 320px para todos los dispositivos
